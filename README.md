# 📦 Gerenciador de Inventário RPG

Este projeto é um sistema de gerenciamento de inventário para RPG desenvolvido em **Java** com **Swing** para a interface gráfica.

## ✨ Funcionalidades

- Adicionar itens ao inventário
- Atualizar itens existentes
- Remover itens
- Suporte a itens equipáveis
- Salvamento e carregamento dos dados em arquivo CSV

## 🛠️ Tecnologias

- Java
- Java Swing
- Persistência de dados em CSV
- Programação Orientada a Objetos (POO)
